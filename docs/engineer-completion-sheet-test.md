# 5번/6번 묶음 테스트 문서

최종 업데이트: 2026-05-28

## 5번. 시공엔지니어 완료보고 흐름 테스트

### 사전 조건

1. `협력사데이터` 시트에 engineer 계정이 있어야 합니다.
2. engineer 계정의 F열 권한은 `engineer`, G열 사용여부는 `승인`이어야 합니다.
3. 월 시트의 테스트 현장 V열 시공엔지니어가 engineer 계정의 B열 이름과 같아야 합니다.
4. 테스트 현장은 관리자 잠금 상태가 아니어야 합니다.

### engineer 로그인 테스트

1. 파트너 포털에서 engineer 계정의 B열 이름으로 로그인합니다.
2. 임시 비밀번호 `0000`이면 강제 비밀번호 변경 화면이 표시되어야 합니다.
3. 새 비밀번호 변경 후 다시 로그인합니다.
4. engineer 권한에서는 본인에게 배정된 현장만 보여야 합니다.
5. partner 전용 지급시공비는 engineer 화면에 표시되지 않아야 합니다.

### 사진등록 테스트

1. engineer 계정으로 현장 상세 모달을 엽니다.
2. `사진등록` 버튼을 누릅니다.
3. `완료사진` 카테고리를 선택합니다.
4. 이미지 파일 1개 이상을 업로드합니다.
5. 업로드 성공 후 화면의 완료사진 개수가 증가해야 합니다.
6. Drive 현장 폴더의 `완료사진` 하위 폴더에 파일이 저장되어야 합니다.

### 중요 이력 등록 테스트

1. 현장 상세 모달에서 `이력등록` 버튼을 누릅니다.
2. 현장 특이사항 또는 완료보고 특이사항을 입력합니다.
3. 등록 후 월 시트 Y열 `현장 중요 이력`에 내용이 누적되어야 합니다.
4. 이력에는 날짜/시간과 작성자명이 함께 들어가야 합니다.

### 완료보고 테스트

1. 완료사진을 1장 이상 등록한 상태에서 `완료보고` 버튼을 누릅니다.
2. 월 시트 S열 진행상태가 `시공완료`로 변경되어야 합니다.
3. 화면에서도 상태가 `시공완료`로 갱신되어야 합니다.
4. 완료사진이 없으면 `완료사진을 1장 이상 등록한 뒤 완료보고할 수 있습니다.` 메시지가 나와야 합니다.

### 권한/잠금 테스트

1. 다른 engineer에게 배정된 현장은 보이지 않아야 합니다.
2. 관리자 잠금 현장은 사진등록, 이력등록, 완료보고가 차단되어야 합니다.
3. 삭제 상태 현장은 처리할 수 없어야 합니다.

## 6번. 실데이터 테스트용 시트 준비

### 협력사데이터 시트 준비

아래 형태로 테스트 계정을 준비합니다.

| A 대리점명 | B 이름/기사명 | C 연락처 | D 사업자번호 | E 비밀번호 | F 권한 | G 사용여부 | H 비밀번호상태 |
|---|---|---|---|---|---|---|---|
| 테스트협력사 | 테스트협력사 | 010-0000-0000 |  | 0000 | partner | 승인 | 임시 |
| 테스트협력사 | 테스트기사 | 010-1111-2222 |  | 0000 | engineer | 승인 | 임시 |

### 월 시트 테스트 현장 준비

월 시트에서 테스트 현장 1건을 정하고 아래 열을 확인합니다.

| 열 | 의미 | 테스트 기준 |
|---|---|---|
| S | 진행상태 | 협력사 배정 후 `협력사배정완료` 또는 배정 전 상태 |
| U | 협력사 | `테스트협력사` |
| V | 시공엔지니어 | 배정 전 비움 또는 `미배정`, 배정 후 `테스트기사` |
| W | 엔지니어 연락처 | 배정 후 `010-1111-2222` |
| Y | 현장 중요 이력 | 이력등록 후 내용 누적 |
| Z | 폴더 URL | 사진 업로드 시 필요, 없으면 자동 생성 |
| AA | 사진보기 | 사진 폴더 링크 |
| AD | 관리자 잠금 | 테스트 중 비워두기 |

### 테스트 순서 요약

1. partner 계정으로 로그인합니다.
2. 테스트협력사 현장이 보이는지 확인합니다.
3. 테스트기사로 엔지니어를 배정합니다.
4. engineer 계정으로 로그인합니다.
5. 테스트 현장이 보이는지 확인합니다.
6. 완료사진을 업로드합니다.
7. 중요 이력을 등록합니다.
8. 완료보고를 저장합니다.
9. 월 시트 S/V/W/Y/Z/AA 열 변경을 확인합니다.

## 관련 API

- engineer 로그인: `partnerLogin`
- 현장 조회: `getPartnerJobs`
- 사진 업로드: `uploadPhoto`
- 이력 등록: `addHistory`
- 완료보고: `completeJob`
